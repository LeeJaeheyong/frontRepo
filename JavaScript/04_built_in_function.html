문자열 관련 내장 함수

  • toUpperCase() : 모든 문자를 대문자로 변환
  • toLowerCase() : 모든 문자를 소문자로 변환
  • length : 문자열의 길이를 반환(함수가 아닌 속성)
  • charAt(index) : 지정한 인덱스에 위치한 문자 변환
  • indexOf("문자") : 지정한 문자가 처음으로 등장하는 인덱스 반환
  • includes("문자") : 특정 문자열이 포함되어 있는지 여부를 true/false 로 반환
  • slice(시작인덱스, 끝인덱스) : 문자열 자르기
  • split(구분자) : 문자열을 분할하여 배열로 반환
