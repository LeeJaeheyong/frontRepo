메인 페이지로 이동
배경
  • backgorund-color : 배경 색상
  • backgorund-image : 배경 이미지
  • backgorund-repeat : 배경 이미지 반복
    • repeat : 가로/세로 모두반복(기본값)
    • repeat-x : 가로 방향만 반복
    • repeat-y : 세로 방향만 반복
    • no-repeat : 반복하지 않음(한번만 표시)
    • space : 이미지 사이의 균등한 간격으로 배치
    • round : 이미지 크기를 조정하여 요소에 딱 맞게 배치
  • background-position : 배경 이미지의 위치를 지정
  • background-attachment : 배경 이미지의 스크롤 동작을 제어
