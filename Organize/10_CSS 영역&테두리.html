영역&테두리

메인 페이지로 이동

영역

div 태그

context : width, height 속성을 사용하여 가로/세로 길이를 정할 수 있음

Padding : 컨텐츠와 테두리 사이에 존재하는 영역(padding, padding-top, ...)

border : 테두리 영역(border, border-top, ...)

margin : 테두리 바깥 영역(margin, mragin-top, ...)


테두리 
            선택자 {
                * 테두리 스타일
                border-위치-style: 속성값;
            }

            선택자 {
                * 테두리 두께
                border-위치-width: 속성값;
            }

            선택자 {
                * 테두리의 색상
                border-위치-color: 속성값;(그냥색상,rgb,#ffffff(헥사코드))
            }

            선택자 {
                border : 두께 스타일 색상;
            }

            선택자 {
                * 모서리를 둥글게 해주는 속성
                border-radius : 속성값;
            }

            선택자 {
                * 그림자 효과
                border-shadow : 가로 세로 흐림 번짐 색상;
            }
