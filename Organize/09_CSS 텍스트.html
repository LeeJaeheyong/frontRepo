텍스트 관련

메인 페이지로 이동
정리
  1. font-size : 텍스트의 크기를 결정(px, %, em, rem, vh, vw)
    • px : 지정한 픽셀 만큼의 크기로 고정되는 절대적인 단위
    • % : 부모 요소의 크기와 비교하여 상대적으로 표현하는 단위
    • em : 현재 요소의 폰트 크기와 비교하여 상대적으로 표현
    • rem : 루트요소(일반적으로 <HTML>)의 크기와 비교하여 상대적으로 표현
    • vw : 브라우저의 가로 길이에 따라서 상대적으로 표현
    • vh : 브라우저의 세로 길이에 따라서 상대적으로 표현
  2. text-align : 텍스트의 정렬 방식을 지정.
    • left : 왼쪽정렬
    • center : 가운데정렬
    • right : 오른쪽 정렬
  3. font-style : 텍스트의 스타일을 지정
    • normal : 일반 텍스트(기본값)
    • italic : 기울어진 텍스트
    • oblique : 기울어진 텍스트(일부 폰트에서는 oblique로 지원)
  4. text-decoration : 텍스트에 밑줄, 취소선, 윗줄 등의 효과를 지정
    • none : 일반 텍스트(기본값)
    • underline : 밑줄
    • line-through : 취소선
    • overline : 윗줄
    • underline overline : 밑줄+윗줄
    • underline line-through overline : 밑줄+취소선+윗줄
  5. text-shadow : 텍스트에 그림자 효과를 지정

  6. white-space : 텍스트의 공백과 줄바꿈에 관련된 속성
    • normal : 공백은 반드시 1개만 가능, 자동 줄바꿈 O(기본값)
    • nowarp : 공백은 반드시 1개만 가능, 자동 줄바꿈 X
    • pre : 공백 여러개 가능, 자동 줄바꿈 O
    • pre-line : 공백 1개만 가능, 자동 줄바꿈 O
