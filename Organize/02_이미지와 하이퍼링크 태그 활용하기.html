이미지와 하이퍼링크 태그 활용하기

메인 페이지로 이동
정리
  1. img 태그는 이미지를 보여주기 위한 태그

    2. alt 속성은 이미지가 로드되지 않을때 어떤 이미지인지 확인하기 위한 텍스트를 출력함

    스크린 리더에서 이미지를 읽어주는 역할을 수행함

  2. audio 태그는 mp3와 같은 오디오 파일을 보여주기 위한 태그
  3. video 태그는 avg와 같은 비디오 파일을 보여주기 위한 태그는
  4. a 태그는 다른주소로 이동하는 링크를 생성할 때 사용하는 태그
    5. 네이버로 이동

    target 속성은 크게 두가지로 나눠서 사용함

    a) _black 속성값은 링크를 새 창으로 열어주는 역할

    b) _self 속성값은 링크를 현재 창에서 열어주는 역할(기본값)
